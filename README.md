# デカ盛りコミュー

## サイト概要
びっくりするようなデカ盛り料理を投稿するサイト

### サイトテーマ
ビックリしたデカ盛り料理を検索したり、投稿したりできるコミュニティサイト

### テーマを選んだ理由
食べることが大好きで好きな料理のデカ盛りをしているお店を共有することができる、
アプリがあっても面白いかなと思い作ってみようかなと思いました。

### ターゲットユーザ
デカ盛りが好きな人

### 主な利用シーン
デカ盛りを知りたいと思った時
